# v6.6.11 Updater Bootstrap Audit

Date: 2026-06-06

## Findings

- R-41 needs repository setup guidance before mutating update commands are
  safe to expose.
- tufup target filenames must not contain whitespace and should use the
  `BookmarkOrganizerPro-<version>` target name used by the client adapter.
- Download/apply commands should fail closed until install directory isolation,
  rollback behavior, and user confirmation are covered by tests.
