# v6.6.12 OPDS Export Audit

Date: 2026-06-06

## Research Findings

- The official OPDS Catalog 1.2 specification defines an Atom-based catalog
  format with acquisition feeds.
- The OPDS specifications overview describes OPDS 1.x as Atom/Dublin
  Core-based and OPDS 2.0 as JSON-LD/Web Publication Manifest-based.
- A file export should start with OPDS 1.2 for broad e-reader compatibility
  before adding loopback serving.

## Changes Shipped

- Added `export_opds()` to `bookmark_organizer_pro.services.feed_export`.
- Added OPDS namespace declarations, open-access acquisition links, language
  metadata, tags/categories, and optional self catalog URL.
- Added EPUB/PDF/HTML media type inference for acquisition links.
- Added `opds-export` CLI command.
- Bumped release metadata to v6.6.12.
