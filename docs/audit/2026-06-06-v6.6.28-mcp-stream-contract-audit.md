# v6.6.28 MCP Stream Contract Audit

## Scope

R-15 advanced from live FastMCP progress coverage to explicit stream contract
metadata. Clients can now verify the event payload contract from tool metadata
and final `chat_with_collection_stream` responses.

## Changes

- Added `CHAT_STREAM_CONTRACT_VERSION`.
- Added `CHAT_STREAM_EVENT_TYPES`.
- Added `stream_contract_version` and `stream_event_types` to final
  `chat_with_collection_stream` payloads.
- Added Bookmark Organizer metadata keys for the stream contract to raw MCP and
  FastMCP tool catalogs.

## Compatibility

- Existing payload fields are preserved.
- The public tool arguments are unchanged.
- Raw MCP clients still receive JSON event payloads.
- FastMCP clients still receive progress notifications when a progress handler
  is provided.
