the future desktop canvas view can render.

- Added bookmark graph construction linking bookmark nodes to tag, category,

- Added `graph-export` CLI with output path and bookmark limit options.

- Existing bookmark records are unchanged.
- Graph export is derived from the current bookmark collection and writes a
  standalone JSON artifact.
- The default export path remains under the app exports directory.
