# v6.6.9 Updater Policy Audit

Date: 2026-06-06

## Research Findings

- `py -3.12 -m pip index versions tufup` confirmed `tufup` 0.10.0 is the
  current release.
- `py -3.12 -m pip index versions tuf` confirmed `tuf` 7.0.0 is the current
  release.
- R-41 should stay opt-in because update repositories and binary replacement
  are high-trust operations.

## Changes Shipped

- Added `bookmark-organizer-pro[updates]` with `tufup>=0.10,<0.11`.
- Added `bookmark_organizer_pro.services.updates` with `UpdateManager`,
  `UpdatePolicy`, `UpdateStatus`, optional dependency readiness, and dotted
  version comparison.
- Persisted update policy in `update_config.json` under the app data directory.
- Required HTTPS for update metadata and target repository URLs.
- Added `updates status`, `updates check`, and `updates configure` CLI commands.
- Bumped release metadata to v6.6.9.
