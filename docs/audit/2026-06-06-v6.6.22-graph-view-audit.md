# v6.6.22 Graph View Audit

Date: 2026-06-06
Scope: R-22 completion, desktop graph canvas

## Objective

Finish R-22 by rendering the bookmark graph model from v6.6.21 in the desktop
application.

## Changes

- Added a Tk-native graph dialog backed by the bookmark graph service.
- Rendered bookmarks, tags, categories, and domains as graph nodes with edges.
- Added canvas pan/zoom.
- Added node selection details.
- Added double-click opening for bookmark nodes.
- Added Graph JSON export from the dialog.
- Added Graph View entry points to the Tools menu and command palette.

## Compatibility

- Existing graph CLI export remains unchanged.
- The graph dialog derives from the current in-memory bookmark collection and
  does not mutate bookmark records.
- Empty libraries show a toast instead of opening an empty graph.

## Follow-Up Candidates

- Add graph filters for tags, categories, and current search results.
- Add graph density presets for very large libraries.
- Add saved graph snapshots if users rely on graph exports.
